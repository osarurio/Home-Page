### ギアシミュレーターの概要
こんにちは！　ギアシミュレーターにようこそ！！
## 遊び方
UIは次のようになっています。
ライブラリから歯車をエディターにドラッグアンドドロップし、エディターに置きます。
エディターの歯車にスナップをして嚙合わせることができます。
「Run」を押すと、あらかじめ設定された速さでモーターギアが回転します。
# 使用できる歯車
　**モーターギア**
　　モーターに接続された一定の速さで回転するギアです。　
  　回転の速さはメニューから変更することができます。
   
  **スタンダードギア**
  　普通のギアです。
   
 **2段ギア**
　大きい歯車と小さい歯車がくっついたギアです。
 このギアで、回転数とトルクを調整できます。
 
  **ラチェットギア**
  　指定の方向のみ回転し、逆方向の回転は無視します。
